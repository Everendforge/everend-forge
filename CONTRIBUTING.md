# Contributing

Everend Forge is split across multiple repositories. Start by choosing the repo that matches the change.

- Format or compatibility changes belong in spec.
- Worldbuilding app changes belong in worldnotion.
- Branching narrative editor changes belong in pathbranching.
- Runtime and engine adapter work belongs in plugins or a future engine-specific repo.
- Suite-level roadmap and project documentation belongs here.

When changing a shared contract, update spec first and then update affected apps/plugins in separate pull requests.
