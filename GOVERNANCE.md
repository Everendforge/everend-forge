# Governance

Everend Forge is currently stewarded in the Everendforge GitHub organization, with SoleipDreams as the founding steward.

Early governance is intentionally lightweight:

- Keep specs portable and app-agnostic.
- Prefer open formats over proprietary storage.
- Avoid moving breaking changes into apps before the spec documents them.
- Keep personal or private narrative content out of public examples.

A more formal governance model can be added once there are external contributors.
